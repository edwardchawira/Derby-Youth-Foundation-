# Cursor Rules Directory

This directory contains custom rules and instructions for Cursor IDE.

## Purpose

Files in this directory can be used to:
- Define project-specific coding standards
- Set up custom instructions for AI assistance
- Configure behavior patterns for the codebase
- Add context about your project structure and conventions

## Usage

Cursor will automatically read files from this directory to understand your project's rules and preferences.

## Example Files

You can create files like:
- `coding-standards.md` - Coding conventions and best practices
- `project-context.md` - Project overview and architecture
- `component-patterns.md` - Component design patterns
- `api-guidelines.md` - API design and usage guidelines

---

**Note**: This directory is for Cursor-specific rules. For general project documentation, use the root directory.
