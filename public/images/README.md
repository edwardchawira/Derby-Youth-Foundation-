## Installation Steps

1. **Locate your original WhatsApp images** (5 files total)
2. **Rename each image** according to the mapping table above
3. **Replace the placeholder files** in `/public/images/` directory
4. **Restart the dev server** if running (automatic rebuild)

## Image Optimization Tips

- Use JPEG format for photos (better compression)
- Recommended quality: 80-85% for web optimization
- Ensure images are properly oriented
- Consider using tools like TinyJPG or ImageOptim before upload

## Fallback Behavior

The site currently has placeholder files. Once you replace them with actual photos:
- Images will load immediately
- No code changes needed
- Automatic responsive scaling applied
